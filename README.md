# Dívida Shadow - Como Usar

## O que é o Dívida Shadow?
Um app para controle de pensão alimentícia com estilo cyberpunk.

## Como usar no celular

### Opção 1: Usar como App (PWA)
1. Acesse o site pelo celular
2. No Chrome: Menu > "Adicionar à tela inicial"
3. No Safari iOS: Compartilhar > "Adicionar à Tela de Início"

### Opção 2: Hospedar na web (GitHub Pages)

#### Passo 1: Criar conta no GitHub
1. Acesse https://github.com
2. Crie uma conta gratuita

#### Passo 2: Criar um repositório
1. Clique em "New repository"
2. Nome: `divida-shadow`
3. Público
4. Clique em "Create repository"

#### Passo 3: Enviar arquivos
1. No seu computador, vá na pasta `C:\Projects\DividaShadow\`
2. Você precisa enviar 2 arquivos:
   - `index.html`
   - `manifest.json`
3. No GitHub, clique em "uploading an existing file"
4. Arraste os 2 arquivos
5. Clique em "Commit changes"

#### Passo 4: Ativar GitHub Pages
1. Vá em Settings > Pages
2. Em "Source", selecione "Deploy from a branch"
3. Em "Branch", selecione "main" e "/ (root)"
4. Clique em Save
5. Aguarde 1-2 minutos

#### Passo 5: Acessar o app
1. O link será algo como: `https://seu-usuario.github.io/divida-shadow/`
2. Compartilhe esse link no celular

## backup dos Dados

### Exportar (salvar)
1. No app, clique em "Ver Histórico"
2. Clique em "💾 Exportar Dados"
3. Salve o arquivo `.json` em um local seguro

### Importar (restaurar)
1. Clique em "📂 Importar Dados"
2. Selecione o arquivo `.json` salvo anteriormente

## Dicas
- Os dados ficam salvos no navegador
- Use "Exportar Dados" regularmente para fazer backup
- Ao mudar de dispositivo, use "Importar Dados" para restaurar
